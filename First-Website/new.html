Hardware Terminology

input device:

An input device is a device that could turn our human languages into computer language(which is 1 and 0)in order for us to communicate and control the computer

output deivce:

An output device is the opposite side of input device. Input device turn human languages into computer language(1 and 0), output device turn computer language into human languages.

Central Processing Unit(CPU):

A Central Processing Unit is a device that run the code of the computer. If input and output devices are the one which make the command in a computer;Central Processing Unit is the one that run those commands in a computer.

Arithmetic Logic Unit(ALU):

An Arithmetic Logic Unit is a circuit that runs mathematical calculation on integar binary numbers inside a computer.

Control Unit:

A Control Unit is a unit that gives commands to input and output devices or Centrol Processing Unit m,which I mentioned above, to tell them what they shouldl do. It is used as a unit to carry commands to other devices or units to make sure they receive the command correctly.

Random Access Memory:

A Random Access Memory is a computer memory that used to store data and allows the user to change it any time.

Optical Memory:

An Optical Memory(or storage) is a memory that uses laser beams to store memories. It stores the memory by making patterns that could be read by the help of the laser light.

Magnetic Memory:

Magnetic Memory is a memory that uses cores to store memory and access them by using electric to magnetize them.

Flash Memory:

A Flash Memory is a memory that could be stored when the computer dies and restored when the computer restarts.

Logic Gates:

A Logic Gate is a circuit that has one or more input device and only one output device. depending on the signal it receives from the input device, it gives the result with and/or/not.
